Close Gap 1 from 05-UAT.md: Change all 9 remaining single-mode dimensions to multi-select, and add mutual exclusion (suppresses) for clearly conflicting option pairs.

Purpose: User reported that dimensions like use_case, subject, scene, lighting, etc. should all allow multi-selection — only style, constraints, and format were multi. The 9 remaining dimensions all defaulted to single-select, preventing users from combining options within a dimension. This fix makes all dimensions multi-select and uses the existing `suppresses` field on OptionItem to handle genuinely incompatible combinations.

Output: Updated worktype config with 9 mode changes, updated defaults array, and suppresses entries on conflicting option pairs across 4 option catalog files.
</objective>
