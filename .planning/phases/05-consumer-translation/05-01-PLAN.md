**Step 1:** Add `getOptionsByConsumerTerm` to `src/lib/prompt/registry/option.registry.ts`.

Add this function at the end of the file (after `getTargetsForOption`, before the final empty line). It follows the same pattern as other registry functions:

```typescript
/** Query options by consumer-facing vocabulary term. Used for quick-entry tag selection.
 *  Returns all options whose consumerTerms array includes the given term. */
export function getOptionsByConsumerTerm(term: string): OptionItem[] {
  const results: OptionItem[] = [];
  for (const option of optionItemMap.values()) {
    if (option.consumerTerms?.includes(term)) {
      results.push(option);
    }
  }
  return results;
}
```

The function uses the existing `optionItemMap` import (already imported from `./state` at line 1). No new imports needed. The `OptionItem` type import (line 1) already covers the return type.
