**Completeness rule (1 rule):**
1. **No subject selected** — Check `selections["subject"]` is empty/undefined. Warning (per D-02 #1): "未指定主体 — 主体是图片生成最重要的维度，建议添加" / "No subject specified — subject is the most critical dimension for image generation. Consider adding one."

**Conflict rules (6 rules), each with source comments citing research (PITFALLS.md Pitfall 4: false confidence from unvalidated combinations):**

For each conflict rule, extract the option IDs from `selections["art_style"]`, `selections["color_palette"]`, `selections["lighting"]` (these are question IDs from the image_prompt work type). Option IDs follow the pattern `{optionSetId}:{optionId}` such as `image_art_style:photorealistic`.

2. **Photorealistic + anime styles** — Check `selections["art_style"]` includes both `image_art_style:photorealistic` AND `image_art_style:anime_manga`. Warning (per D-02 #2): "同时选择了写实和动漫风格 — 两套视觉语言冲突，建议只保留其一" / "Both photorealistic and anime style selected — these visual languages conflict. Consider keeping only one."

3. **B&W monochrome + vibrant color** — Check `selections["color_palette"]` includes both `image_color_palette:monochrome` AND `image_color_palette:vibrant`. Warning (per D-02 #3): "黑白单色和高饱和色彩不能同时生效 — 两种色彩方案互斥，建议只保留其一" / "Black-and-white monochrome and vibrant color cannot coexist — these color schemes are mutually exclusive. Consider keeping only one."

4. **Watercolor + sharp focus** — Check `selections["art_style"]` includes `image_art_style:watercolor` AND any of `image_art_style:photorealistic`, `image_art_style:photorealistic_render`, `image_art_style:macro_photography`. Warning (per D-02 #4): "水彩风格通常使用柔和焦点，与锐焦/写实风格冲突 — 建议只保留其一" / "Watercolor style typically uses soft focus and conflicts with sharp/realistic rendering. Consider keeping only one."

5. **Multiple art mediums** — Check `selections["art_style"]` includes 3+ of the art medium category option IDs (oil_painting, watercolor, pencil_sketch, acrylic_impasto, digital_painting, ink_wash, line_art, marker_drawing, pastel_drawing, woodcut_print, collage, paper_cut, embroidery, mosaic). Warning (per D-02 #5): "选择了多种创作媒介（{list the labels}）— 建议只保留一个主要媒介" / "Multiple art mediums selected ({list labels}) — consider keeping only one primary medium."

6. **Flat vector + volumetric lighting** — Check `selections["art_style"]` includes `image_art_style:vector_flat` AND `selections["lighting"]` includes any volumetric lighting option IDs (need to check actual option IDs). Warning (per D-02 #6): "扁平矢量风格不需要体积光 — 建议移除光线选择或改用其他风格" / "Flat vector style does not require volumetric lighting — consider removing lighting choices or switching art style."

7. **3D rendered + hand-drawn** — Check `selections["art_style"]` includes `image_art_style:3d_render` AND any hand-drawn option IDs (`image_art_style:pencil_sketch`, `image_art_style:line_art`, `image_art_style:marker_drawing`, `image_art_style:pastel_drawing`). Warning (per D-02 #7): "3D渲染和手绘感是两种不同的创作方式 — 建议只保留其一" / "3D render and hand-drawn are different creative approaches — consider keeping only one."
