# 可控提示词向导 v1.1：图片提示词生成现已上线

经过几个版本的迭代，可控提示词向导迎来了一个重要里程碑：从即日起，你不仅可以用它生成视频提示词，还可以用它生成专业的图片提示词。

我们一直相信，好的提示词不应该需要背诵术语，也不应该依赖付费 API 逐字调整。v1.1 延续了这一理念——你只需要在界面上做选择题，系统就会自动将你的选择拼接成逗号分隔的自然语言图片提示词，一键复制即可使用。生成的提示词兼容 Midjourney、Stable Diffusion、Flux、DALL-E 等所有主流图片生成模型，不需要针对不同工具单独调整格式。

这一次，我们为图片提示词准备了非常全面的选项体系：涵盖用途、主体、场景、构图、镜头、光影、画幅、色彩、风格、氛围、季节/时间、景别、画质和艺术家参考共 14 个维度，总计 272 个专业选项。此外还有 80 个消费者美学快捷词——像"高级感""ins风""莫兰迪色"这样的标签，每一个都对应一组高质量的专业提示词组合，让你不需要懂任何术语就能生成效果出众的图片提示词。

最重要的是：整个工具完全在你的浏览器本地运行。不需要注册账号，不需要调用任何付费 API，你的每一个选择都只保存在你自己的电脑上。这是真正私密、免费、永远可用的提示词工具。

打开可控提示词向导，点击顶部的"图片"标签，试一下你的第一组图片提示词吧。

# 图片提示词实战指南：从零到可复制提示词

如果你从来没有写过图片提示词，这篇指南会用十分钟带你走完一个完整流程。你不需要懂任何术语，也不需要参考外部教程——只需要跟着步骤做选择，让工具帮你完成剩下的工作。

**第一步：切换工作类型**

打开可控提示词向导，点击页面顶部的"图片"按钮。界面会自动切换到图片提示词模式，你会看到全新的 14 维度选项面板，每个维度都有一组卡片式的可选项。

**第二步：选择使用场景**

工具提供了 18 个图片使用场景，每个场景都预设了推荐的话题和视觉方向。作为电商运营，你需要一组产品图，就选择"产品摄影"；作为自媒体博主，你需要社交配图，就选"社交媒体内容"。

**第三步：确定拍摄主体**

选择你要展示的核心对象。假设我们要为电商详情页拍一个白色陶瓷杯，在主体选项中勾选"白色陶瓷杯"。工具会自动关联相关的主体描述词。

**第四步：设置场景背景**

根据内容调性选择拍摄环境。"室内工作室"适合质感优先的产品图，"户外自然光"适合生活方式类内容。产品摄影建议选室内工作室，以获得干净无干扰的背景。

**第五步：选择艺术风格**

这是定调的一步。你可以从专业风格中挑选（如"写实摄影""极简风""油画质感"），也可以使用消费者美学快捷标签直接选你想要的"感觉"。比如"极简风"——一个标签就定义了整体视觉方向。"高级感""ins风""大片感"这些标签，每一个背后都关联着多条专业提示词，你完全不用关心技术细节。

**第六步：打开高级选项，精细化调整**

点击"高级选项"开关，更多控制维度会出现。为你的陶瓷杯选择"莫兰迪色"作为整体色调，选择"温暖宁静"作为氛围基调，选择"上午阳光"定义时间与光线方向。这些选项的组合会把你的提示词从"能用"变成"精良"。

**第七步：查看并复制提示词**

右侧输出面板实时显示你的所有选择拼接后的结果。中文提示词、英文提示词、JSON 结构化数据和 Markdown 格式副本共四种导出方式任你选择。点一下"复制中文"，提示词就进剪贴板了。

**第八步：粘贴到图片生成工具**

打开 Stable Diffusion WebUI、Midjourney 或 Flux，将提示词粘贴到输入框。逗号分隔的自然语言格式可以直接使用，不需要任何手动调整或参数微调。

**一个完整的例子**

当你依次选择了"产品摄影 + 白色陶瓷杯 + 室内工作室 + 上午阳光 + 极简风 + 莫兰迪色"，输出面板会实时生成类似这样的提示词：

> 产品摄影，白色陶瓷杯，放置在室内工作室中，上午柔和的自然光从窗户洒入，极简主义风格，莫兰迪色系调色板，温暖宁静的氛围，干净简洁的构图，高质量细节

你不需要写一个字，所有内容都来自你的选择。

**两个实用技巧**

1. **URL 分享**：你的所有选择都会实时编码到浏览器地址栏中。直接把 URL 复制发给同事或客户，对方打开链接就能看到和你完全相同的配置，不需要任何额外说明。
2. **状态自动保存**：关闭标签页再重新打开，你上次的所有选择会自动恢复。工具使用 localStorage 在本地保存了工作状态，完全不需要注册登录——这是你自己的工具，数据也只在你自己的设备上。

现在打开可控提示词向导，切换到图片模式，试一组你自己的提示词吧。也许你会发现，原来写提示词这件事，就只是做几道选择题而已。
